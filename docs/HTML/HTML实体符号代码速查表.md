# [HTML实体符号代码速查表](https://www.cnblogs.com/kiter/archive/2011/08/05/2128309.html)

**1.特色的**

| ©    | &copy;   | ©    | 版权标志                                                 |
| ---- | -------- | ---- | -------------------------------------------------------- |
| \|   |          | \|   | 竖线，常用作菜单或导航中的分隔符                         |
| ·    | &middot; | ·    | 圆点，有时被用来作为菜单分隔符                           |
| ↑    | &uarr;   | ↑    | 上箭头，常用作网页“返回页面顶部”标识                     |
| €    | &euro;   | €    | 欧元标识                                                 |
| ²    | &sup2;   | ²    | 上标2，数学中的平方，在数字处理中常用到，例如：**1000²** |
| ½    | &frac12; | ½    | 二分之一                                                 |
| ♥    | &hearts; | ♥    | 心型，用来表达你的心                                     |
