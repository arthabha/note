linux下编译安装phalcon

1、下载源码包

运行命令： git clone --depth=1 git://github.com/phalcon/cphalcon.git

进入目录 运行命令： cd build

执行安装文件 提示需要指定PHP的配置文件

运行命令： ./install --php-config /usr/local/php/bin/php-config --phpize /usr/local/php/bin/phpize

安装过程可能中断，提示需要re2c 然后去下载 运行命令： wget https://jaist.dl.sourceforge.net/project/re2c/old/re2c-0.13.4.tar.gz 解压，编译安装。

然后重新再执行phalcon的 运行命令： ./install --php-config /usr/local/php/bin/php-config --phpize /usr/local/php/bin/phpize

编译完成之后，提示安装成功。

2、修改php.ini 添加phalcon.so扩展

运行命令： vi /usr/local/php/etc/php.ini

添加内容： extension="phalcon.so" 然后保存，重启php。
